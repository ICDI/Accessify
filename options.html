ℹ️ Sobre

Accessify v1.0.0

Um plugin do Chrome que transforma páginas web em acessíveis seguindo as diretrizes WCAG 2.1, com foco em usuários cegos ou de baixa visão.

Recursos:
  • Análise automática de conformidade WCAG
  • Geração de textos alternativos com IA
  • Otimização de estrutura HTML
  • Melhorias de contraste de cores
  • Adição de ARIA labels
Aviso Legal:
Este plugin é fornecido "como está". Sempre revise as mudanças de acessibilidade antes de aplicá-las em produção. Recomenda-se testar com usuários reais e ferramentas de acessibilidade profissionais.

Recursos úteis:
• Diretrizes WCAG 2.1
• OpenRouter
• Web Accessibility Initiative (WAI)
