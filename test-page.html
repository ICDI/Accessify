Seção 5: Alertas de Legibilidade

Este é um bloco propositalmente muito longo para disparar o alerta de texto justificado no plugin. O objetivo aqui é reproduzir um cenário em que um parágrafo extenso foi configurado com alinhamento totalmente justificado, criando espaçamentos irregulares entre palavras, rios de branco e piorando a leitura para pessoas com baixa visão, dislexia ou dificuldades cognitivas. Como o algoritmo desejado considera elementos do tipo parágrafo, divisão ou célula de tabela com mais de quinhentos caracteres e text-align: justify, este texto continua se estendendo com conteúdo repetido para ultrapassar com folga esse limiar, permitindo validar que a correção automática remove o alinhamento justificado e retorna o texto para uma apresentação mais estável e previsível na tela.

Problemas Identificados Nesta Página

Este página contém propositalmente os seguintes problemas de acessibilidade:
  • ❌ Sem H1 (título principal)
  • ❌ Estrutura de títulos ruim (H3 → H5)
  • ❌ Imagens sem alt text
  • ❌ Imagem com alt text muito curto
  • ❌ Links com texto genérico
  • ❌ Formulário sem labels
  • ❌ Tabela sem headers semânticos
  • ❌ Contraste de cores baixo
  • ❌ Texto longo justificado
  • ❌ Texto muito pequeno
  • ❌ Conteúdo com aria-hidden
  • ❌ Title redundante
  • ❌ Alt redundante com texto adjacente
  • ❌ Div com onclick (não semântico)
  • ❌ Sem indicador de foco visual

Como Testar o Plugin

  1. Clique no ícone ♿ na barra de ferramentas
  2. Clique em "🔍 Analisar Acessibilidade"
  3. Aguarde a análise ser concluída
  4. Revise os problemas encontrados
  5. Clique em "✨ Aplicar Otimizações"
  6. Observe as mudanças na página
