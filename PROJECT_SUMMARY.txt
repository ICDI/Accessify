DESCRIÇÃO DO PROJETO
================================================================================

Um plugin poderoso do Google Chrome que transforma páginas web em acessíveis
seguindo as diretrizes WCAG 2.1, com foco especial em usuários cegos ou de
baixa visão.

O plugin utiliza inteligência artificial (OpenRouter) para analisar páginas,
gerar textos alternativos para imagens e recomendar otimizações de acessibilidade.

FUNCIONALIDADES PRINCIPAIS
================================================================================

1. ANÁLISE WCAG AUTOMÁTICA
   - Avalia conformidade com WCAG 2.1 (Nível A e AA)
   - Identifica 10+ critérios de acessibilidade
   - Classifica problemas como críticos ou avisos

2. GERAÇÃO DE ALT TEXT COM IA
   - Usa OpenRouter para análise de imagens
   - Gera descrições automáticas e descritivas
   - Suporta múltiplas imagens

3. OTIMIZAÇÕES INTELIGENTES
   - Melhora estrutura HTML semântica
   - Otimiza contraste de cores
   - Adiciona ARIA labels apropriados
   - Garante navegação por teclado

4. INTERFACE AMIGÁVEL
   - Mostra etapas de otimização em tempo real
   - Permite revisar mudanças antes de aplicar
   - Suporta otimização automática
   - Painel de configurações intuitivo

================================================================================
INSTALAÇÃO RÁPIDA
================================================================================

1. Abra chrome://extensions/
2. Ative "Modo de desenvolvedor"
3. Clique em "Carregar extensão não empacotada"
4. Selecione a pasta do plugin
5. Acesse as configurações e insira sua chave de API OpenRouter
6. Pronto! Clique no ícone ♿ para começar

Para instruções detalhadas, veja INSTALLATION.md

================================================================================
COMO USAR
================================================================================

ANÁLISE MANUAL:
1. Navegue até uma página web
2. Clique no ícone ♿ na barra de ferramentas
3. Clique em "Analisar Acessibilidade"
4. Revise os problemas encontrados
5. Clique em "Aplicar Otimizações"

OTIMIZAÇÃO AUTOMÁTICA:
1. Acesse as configurações do plugin
2. Ative "Otimizar automaticamente todas as páginas"
3. O plugin analisará cada página visitada

================================================================================
ARQUITETURA
================================================================================

O plugin é estruturado em 3 camadas:

1. CONTENT SCRIPT (content.js)
   - Executa no contexto da página
   - Avalia conformidade WCAG localmente
   - Aplica otimizações à página

2. BACKGROUND SERVICE WORKER (background.js)
   - Gerencia integração com OpenRouter
   - Armazena configurações
   - Comunica com popup e content script

3. INTERFACE DO USUÁRIO
   - Popup: Interface principal
   - Options: Página de configurações

Fluxo de Dados:
Página → Content Script → Background → OpenRouter API → Recomendações

================================================================================
INTEGRAÇÃO COM OPENROUTER
================================================================================

API ENDPOINT: https://openrouter.io/api/v1/chat/completions

MODELOS DISPONÍVEIS:
- Auto (Recomendado) - OpenRouter escolhe o melhor
- GPT-4 Turbo - Mais avançado
- GPT-3.5 Turbo - Rápido e econômico
- Claude 3 Opus - Poderoso
- Mistral Large - Código aberto

SEGURANÇA:
- Chave de API armazenada localmente
- Nunca enviada para servidores não autorizados
- Conteúdo da página processado apenas para otimização
